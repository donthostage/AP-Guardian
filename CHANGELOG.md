# Changelog AP-Guardian

## Версия 1.1.0 - Улучшения и новые функции

### Добавлено

1. **Детектор брутфорс атак**
   - Обнаружение множественных неудачных попыток подключения
   - Мониторинг критических портов (SSH, FTP, HTTP, MySQL, PostgreSQL)
   - Настраиваемые пороги и временные окна

2. **Система статистики**
   - Сбор статистики по пакетам
   - Статистика угроз и блокировок
   - Топ атакующих IP адресов
   - Статистика по портам
   - Время работы системы

3. **Система уведомлений**
   - Email уведомления о угрозах
   - Webhook уведомления
   - Выполнение скриптов при обнаружении угроз
   - Защита от спама уведомлений (cooldown)

4. **Улучшенная обработка DDoS атак**
   - Отслеживание IP источников атак
   - Более точное определение атакующих IP
   - Улучшенная статистика по источникам

5. **Улучшенный захват пакетов**
   - Более эффективная обработка через raw socket
   - Улучшенная обработка ошибок
   - Поддержка асинхронного захвата через scapy

### Улучшено

1. **Обработка ошибок**
   - Более надежная обработка исключений
   - Логирование ошибок без прерывания работы

2. **Производительность**
   - Оптимизация обработки пакетов
   - Улучшенная очистка старых данных

3. **Мониторинг**
   - Расширенная статистика в API
   - Детальная информация о работе системы

### Исправлено

1. Исправлена обработка ARP пакетов в главном модуле
2. Улучшена обработка сигналов остановки
3. Исправлены проблемы с блокировкой IP при DDoS атаках

## Версия 1.0.0 - Первый релиз

### Основные функции

1. Детектор ARP Spoofing
2. Детектор DDoS атак (SYN Flood, UDP Flood, ICMP Flood)
3. Детектор сканирования сети (горизонтальное и вертикальное)
4. Автоматическое управление Firewall
5. LuCI веб-интерфейс
6. API для экспорта статуса
